Automatic contacts ,call logs and sms backup

4 minute read

Automatic call history sms backup

One of the basic functions of Google services that are are part of the its core services is automatic back up and restore of our contacts, messages and call history.

You change your phone or swap a rom , put your Gmail in and boom it is ready.

Here we will see how to replace that foss stuff

we’ll use nextcloud, an open source personal cloud platform selfhosted on a raspberry pi or using a partner free open source provider (I tried opsone and it is excellent) also some android messaging applications that also offer extra security and have options for automated backup which we will take advantage of and upload the local file to our personal encrypted cloud

There are many open foss messaging apps that offer a high level of security and can be backed up locally. But here we will see signal and silence which can also send traditional sms By the way, sms does not offer encryption and is managed by the mobile provider.

Auto sms backup with signal / silence & nextcloud

Initially our applications need a repository to be automatically upgraded by fdroid. Silence and nextcloud are available from the default repository of fdroid but signal is not because it uses firebase to manage push notifications. So we add to fdroid one of the known repos which includes signal here Otherwise if we use microg we can download it from aurora store which will also manage the updates.
