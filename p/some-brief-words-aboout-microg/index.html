some brief words aboout microG

4 minute read

What is it

microG core is a free and open surce clone of Google’s proprietary core libraries modified having most tracking elements removed. Their main function is to disguise themselves and declare to the applications that require google services to work properly that exist and are present.

Services Framework Proxy (GsfProxy) undertakes cloud messaging, ie notifications from applications

Store (Phonesky) undertakes the updates of the applications

What is not

MicroG is not yet an “alternative” gapps package. You will not have the same user experience. However, there are packages that make the transition from a gapps environment to a de-googled environment smoother.

fake store vs modded play store

The modded store provides better compatibility with applications and installs a real google play store with a user experience closer to the gapps environment

The fake store simply states to the applications that are looking for it that it is there but in practice it is a dummy, empty app.
