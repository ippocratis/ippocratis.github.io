sshfs

not everyone has a monitor attached to his raspberry and even if does its always convenient to ssh to it and do stuf from your working machine directly but what about files? terminal is not user friendly to work with files well we’ll mount it to our file system over ssh

install

on raspberry but also on host

apt install sshfs

create a folder on host

mkdir folder_name

and mount your raspberry into that folder

sshfs user_name@port_number: folder_name_on_host

user_name and port_number are obviusely your rpi user and its port on lan

now open your file manager and launch the newly mounted folder with your raspberry files inside it

unmount

fusermount -u folder_name_on_host

if you want to mount the root instead of the home rpi folder

mount root /

sudo sshfs -o allow_other user_name@port_number: folder_name_on_host
